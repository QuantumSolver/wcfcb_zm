Error Analysis

Original Error: "" is not a valid color

Conclusion: This error is NOT coming from the wcfcb_zm app. The error is likely from:

  • Frappe core JavaScript validation
  • Another app or custom script
  • Browser CSS validation
  • A form field or configuration with an empty color value

Note: All color values in wcfcb_zm are properly defined with valid hex codes or CSS variables.
